# 用户指南

## 快速开始

### 安装客户端

1. **下载安装包**
   - 访问项目发布页面
   - 下载对应操作系统的安装包
   - Windows: `.exe` 或 `.msi` 文件

2. **安装程序**
   - 双击安装包
   - 按照安装向导完成安装
   - 安装完成后启动客户端

### 基本使用

#### 一键启动加速

1. 启动游戏加速器客户端
2. 点击主界面的"一键启动"按钮
3. 等待连接状态显示为"已连接"
4. 启动APEX游戏即可

#### 停止加速

1. 点击"停止加速"按钮
2. 等待连接断开
3. 系统代理自动清除

## 功能说明

### 主界面

主界面包含以下功能：

- **一键启动按钮**: 快速启动加速服务
- **停止加速按钮**: 停止加速服务
- **连接状态**: 显示当前连接状态
- **游戏检测**: 自动检测APEX游戏进程

### 状态栏

状态栏显示以下信息：

- **连接状态**: 已连接/未连接/连接中/连接失败
- **延迟**: 实时显示网络延迟
- **上传速度**: 当前上传速度
- **下载速度**: 当前下载速度
- **服务器信息**: 当前连接的服务器

## 使用技巧

### 最佳使用流程

1. **启动加速器**
   - 先启动游戏加速器
   - 点击"一键启动"
   - 等待连接成功

2. **启动游戏**
   - 连接成功后启动APEX
   - 系统会自动检测游戏
   - 游戏流量自动加速

3. **游戏结束**
   - 关闭游戏后
   - 点击"停止加速"
   - 或等待自动断开

### 性能优化建议

1. **网络环境**
   - 使用有线网络优先
   - 关闭其他占用带宽的应用
   - 选择网络负载较低的时间段

2. **系统设置**
   - 关闭不必要的后台程序
   - 更新网卡驱动
   - 检查防火墙设置

## 常见问题

### 连接问题

**Q: 点击"一键启动"后一直显示"连接中"**
A:
- 检查网络连接是否正常
- 确认服务器状态是否正常
- 尝试重新启动客户端

**Q: 显示"连接失败"**
A:
- 检查网络连接
- 确认服务器地址配置正确
- 查看错误提示信息

### 游戏问题

**Q: 游戏无法启动**
A:
- 检查游戏文件完整性
- 确认游戏版本是否支持
- 尝试重启游戏

**Q: 游戏延迟没有改善**
A:
- 检查连接状态是否正常
- 查看延迟数据是否正常
- 尝试更换网络环境

### 系统问题

**Q: 客户端无法启动**
A:
- 检查系统权限
- 确认防火墙设置
- 重新安装客户端

**Q: 系统代理配置失败**
A:
- 以管理员身份运行客户端
- 检查系统代理设置
- 手动清除系统代理

## 高级设置

### 服务器配置

如果需要更改服务器配置：

1. 打开设置面板
2. 找到服务器设置
3. 修改服务器地址和端口
4. 保存设置并重新连接

### 代理设置

系统代理设置说明：

- **代理地址**: 127.0.0.1
- **代理端口**: 7890
- **代理类型**: HTTP/HTTPS

### 日志查看

查看详细日志：

1. 打开设置面板
2. 找到日志设置
3. 点击"查看日志"
4. 日志文件会自动打开

## 故障排除

### 诊断步骤

遇到问题时，按以下步骤诊断：

1. **检查网络连接**
   - 确认网络是否正常
   - 测试服务器连接
   - 检查防火墙设置

2. **检查应用状态**
   - 查看连接状态
   - 检查延迟数据
   - 查看错误信息

3. **查看日志**
   - 打开日志文件
   - 查找错误信息
   - 记录错误代码

4. **重启应用**
   - 关闭客户端
   - 重新启动
   - 尝试重新连接

### 常见错误代码

- **E001**: 网络连接失败
- **E002**: 服务器无响应
- **E003**: 代理配置失败
- **E004**: 游戏进程检测失败
