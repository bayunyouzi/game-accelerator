# 更新日志

## [1.0.0] - 2024-01-01

### 新增
- 初始版本发布
- 支持APEX游戏加速
- 一键启动功能
- 实时状态显示
- 系统代理自动配置
- 游戏进程自动检测
- 延迟监控
- 流量统计

### 技术特性
- 服务器端：Alpine Linux + sing-box + Docker
- 客户端：Tauri + React + TypeScript
- 支持GitHub代码管理
- 支持Zeabur一键部署
- WebSocket实时通信

### 已知问题
- 首期仅支持APEX游戏
- 仅支持Windows系统

### 计划功能
- 支持更多游戏
- 支持多节点选择
- 支持macOS系统
- 添加设置面板
- 优化性能

## 版本说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**：重大功能更新，可能不兼容
- **次版本号**：新增功能，向后兼容
- **修订号**：bug修复，向后兼容
